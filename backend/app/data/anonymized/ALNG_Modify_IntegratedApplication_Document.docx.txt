# Instructions

## Pre-modification Steps

Refer “[Anonymized Customer]” to integrate PaaS (VBCS) application in SaaS.

## Modify PaaS application integration

- Login to Fusion
- In home page, on top right hand side click on UserName icon and select “Setup and Maintenance” item from drop down menu.
- Then select “Manage Setup content” task from Task List available on right hand side of page.
- Then select “Manage Integration of Additional Application”.
- It will navigate to “Manage Integration of Additional Application” page. In this page select  “[Anonymized Customer]” application and click on edit option.
- In edit Application Integration page, replace existing Full URL with new one.
In our case the existing URL is referring to PROD environment which is below
https://[Anonymized Customer]-oraclecloudinteg.integration.ocp.oraclecloud.com/ic/builder/rt/[Anonymized Customer]/live
- Replace PROD URL with DEV1 URL which is as follows.
https://[Anonymized Customer]-oraclecloudinteg.integration.ocp.oraclecloud.com/ic/builder/rt/[Anonymized Customer]/live
Note:
For UAT use the OIC UAT URL respectively
https://[Anonymized Customer]-oraclecloudinteg.integration.ocp.oraclecloud.com
- After replacing URL click on Apply button and then click on Save and Close button.
